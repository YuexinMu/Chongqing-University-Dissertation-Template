# CQU 大数据与软件学院学术学位论文模板（LaTeX）

本仓库是在 [CQU_Thesis2024 模板](https://github.com/yinguoweiOvO/CQU_Thesis2024) 的基础上，根据教务老师提供的《（学术学位模板）大数据与软件学院博士、硕士论文格式范本-综合类 - 批注版 (2025.6).docx》进行修改和适配，仅供参考使用。

> 由于规范文档为 Word 格式，LaTeX 实现中可能存在一定程度近似或折中处理，欢迎提出 Issue/PR 共同完善。

> ⚠️ 重要说明：
> - 本模板 **并非官方模板**，也不保证与后续最新规范完全一致；
> - 使用前请务必对照学院/研究生院最新论文写作与排版要求，自行核对与调整；
> - 如与学院最新 Word 范本或教务老师要求存在差异，请以 **学院最新正式文件** 为准。

## 仓库结构

典型目录结构如下（示意）：

```text
.
├── main.tex              # 论文主文件（入口）
├── cquthesis.cls         # 论文类文件（样式定义，基于原模板修改）
├── config/               # 配置相关文件（如个人信息、封面信息等）
├── chapters/             # 各章节 tex 文件
│   ├── chap01-intro.tex
│   ├── chap02-related.tex
│   └── ...
├── figs/                 # 图片资源
├── references.bib        # 参考文献数据库（.bib 文件）
├── templates/            # 可能的示例文件、额外模板
└── README.md             # 说明文档（即本文件）
```

实际结构请以仓库当前文件为准。

## 使用说明

### 1. 环境准备

建议使用以下环境编译：

- TeX 发行版：TeX Live 2022 及以上 / MiKTeX 最新版
- 编译引擎：**XeLaTeX**（推荐）
- 推荐编辑器：
  - Overleaf（需按需上传全部文件）
  - VS Code + LaTeX Workshop
  - TeXstudio / TeXworks

### 2. 编译方式

以 `main.tex` 为入口文件，推荐的本地编译流程：

```bash
# XeLaTeX -> BibTeX/Biber -> XeLaTeX -> XeLaTeX
xelatex main.tex
bibtex main.aux   # 或使用 biber，视模板配置而定
xelatex main.tex
xelatex main.tex
```

或使用编辑器提供的一键编译功能，选择 **XeLaTeX** 作为默认引擎。

### 3. 填写个人与论文信息

一般在 `config` 目录或主文件中有集中配置区，用于设置（但是封面、学位独创性申明这些建议直接上传PDF，模版出来的问题可能比较多）。


### 4. 章节与内容组织

在 `chapters/` 目录下按章节拆分 tex 文件，例如：

```tex
% main.tex 中示意
\include{chapters/chap01-intro}
\include{chapters/chap02-related}
...
```

## 注意事项

1. **规范更新风险**  
   本仓库依据的是 2025 年 6 月版的 Word 范本，后续若学院或研究生院发布新的格式要求，本仓库可能未能及时同步。

2. **格式核对责任**  
   使用者需在最终提交前，自己对照：
   - 最新学院/研究生院论文撰写规范
   - 最新 Word 模板与批注要求
   - 导师或学院教务老师的具体指导

## 贡献与反馈

- 欢迎通过 **Issues** 提交：
